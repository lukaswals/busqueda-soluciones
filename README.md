# busqueda-soluciones

Tested on Python 3
The following explanation is written in Spanish. English explanation coming soon.

## Descripcion

Simple código para probar algoritmos de solucion de problemas mediante búsquedas. Se incluyen las siguientes búsquedas:

Desinformadas o a ciegas:
+ Primero en Anchura
+ Primero en Profundidad
+ Costo Uniforme

Informadas:
+ Primero en Profundidad Informadas
+ Best First
+ A\*

## Contenido

+ **sistema.py**  : Incluye la función "main", desde aquí ejecutamos el software.
+ **nodos.py**    : Contiene la definición de dos tipos de Juegos, "Encontrar el camino" y "Problema de las N Reinas".
+ **busqueda.py** : Contiene la definición de los diferentes algoritmos de búsqueda.
